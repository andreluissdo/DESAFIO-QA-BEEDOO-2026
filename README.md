# DESAFIO-QA-BEEDOO-2026

Objetivo: A ferramenta foi criada para cadastra novos cursos e informar em lista quais os cursos cadastrados.
Seus principais fluxos são o o cadastro de curso onde se deve preencher um formulários com as informações do curso que quer se criar e o fluxo de visualização dos cursos onde são apresentados os cursos já criados.
Os pontos mais críticos para testes são as informações a serem inseridas no cadastro do curso para verificar se as informações são válidas e não apresentão má intenção e a apresentação da lista de cursos para verificar se a lista está correta e as informações aparecem conforme planejado.
